# svn

  1. SVN 全称subversion 是一个集中式版本控制工具
  2. 两个区
    • 服务区：用来存储数据的仓库。（创建仓库、创建用户、设置权限等）
    • 用户区：用来写源代码的地方
  3. 操作：

    • checkout

      • 将远程仓库的所有内容下载（复制、克隆）到本地来（只会做一次）

    • update

      • 更新远程仓库的最新的内容（从第二次开始，做多次）

    • commit 提交本地的代码到远程仓库去

    • show log 查看仓库所有代码版本历史记录

    • revert 撤销本地未提交的代码

    • update to reversion 回退到仓库保管的指定版本号

    • 删除本地保存的用户名密码，使用其他用户：鼠标右键 -- 小乌龟 -- setting -- saved Data -- clear all

  4. 问题：冲突
    • 产生原因：多个人在同一个文件修改了同一处不同代码，产生冲突
    • 解决： edit conflicts 编辑冲突 手动解决：删除多余的代码和文件 （注意：不要修改别人的代码）
  5. 分支与合并
    • checkout thunk主干 -- 鼠标右键 -- 小乌龟 -- merge -- 选择第一个 -- 选择合并分支文件夹和版本号（注意两个都必须选） -- 下一步、合并
